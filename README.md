# Image → PDF

A tiny, single-file tool to arrange images on a page and export a PDF.

Drop images onto a Letter/A4/Legal page, then move, resize, rotate, and crop
them freely. Add as many pages as you like and export everything to a PDF.
Handy for things like putting the front and back of an ID card on one page.

## Usage

Open `index.html` in any modern browser. That's it.

- No installation, no build step, no server.
- Runs entirely in the browser — your images never leave your computer.
- The PDF is generated client-side (images are embedded at full resolution).

## Features

- Move, resize (aspect-locked), and free-angle rotate images on the page
- Crop with a full-image cropper
- Multiple pages
- Letter, A4, and Legal paper sizes, portrait or landscape
